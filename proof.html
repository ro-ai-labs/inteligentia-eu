What We'll Prove

A repeatable protocol designed for skeptics: buyers, bar associations, compliance, and auditors.

Offline Privacy

The pilot demo is designed to run with your laptop in airplane mode and no device networking hardware present.

Zero Telemetry

No usage analytics. No “anonymized” collection. The appliance is architected to minimize data transfer risk.

No Sub‑Processors

We plan a processor-free posture: your client data stays local. That means no DPA/SCC maze for cloud inference.

Citations‑First

Every answer is designed to be tied back to local paragraphs with clickable citations for verification.

The Airplane Mode Test

The shortest demo that matters. If it works offline, your data stayed on your desk.

  1. Put your laptop in airplane mode (and/or unplug Ethernet).
  2. Connect Inteligentia via USB‑C (power adapter included).
  3. Load a confidential document set.
  4. Ask a query and verify sources via clickable citations.
Goal: demonstrate no dependency on internet-hosted inference endpoints by our system. This does not assert the host computer is malware-free.

The Privacy Card Ritual

A checklist built for repeatability—something you can run, record, and share internally.

  1. Isolate host: disable Wi‑Fi and unplug Ethernet.
  2. Verify isolation: confirm no default route / internet access.
  3. Connect appliance: USB‑C data + separate power.
  4. Execute workflow: ingest a dossier, search, summarize, draft.
  5. Result: workflow runs end‑to‑end without internet access.
In the pilot, this ritual becomes a printed “Proof of Privacy Card” included in the box.

Processor‑Free Posture (No DPA/SCC Burden)

Designed so regulated professionals don’t need enterprise cloud paperwork just to use AI on confidential work.

  1. No cloud inference: processing is planned to run on the appliance.
  2. No sub‑processors: we do not plan to process client data on RO AI Labs infrastructure.
  3. Updates without data transfer: updates are downloaded by the companion app (when you’re online), verified, then transferred over USB.
For security detail, see Security & Privacy.
